# SI-HYX

Инструмент для скачивания и обработки медиа: загрузка видео/аудио (YouTube, TikTok, Instagram и др. через yt-dlp), перекодирование видео в AV1, обработка звука (нормализация, fade), конвертация изображений (AVIF/WebP/PNG/JPG/ICO), Base64, объединение фото и менеджер промптов. Есть браузерное расширение для отправки ссылок в программу.

## Скачать

Готовую программу бери во вкладке **[Releases](../../releases)** → скачай архив с суффиксом **`-full`** (например `SI-HYX-v0.3.6-full.zip`), распакуй и запусти `SI-HYX.exe`.
Устанавливать ничего не нужно — ffmpeg, yt-dlp и deno уже лежат в папке `bin/` рядом с программой.

> В релизе три файла: `…-full.zip` (код + bin, ставится с нуля), `…-update.zip` (только код, для дельта-обновления) и `manifest.json`. Для первой установки нужен именно **full**; `update`/`manifest` программа использует сама при автообновлении.

## Возможности

- **Загрузчик** — видео/аудио по ссылке, выбор качества, обрезка по таймингам, субтитры, куки.
- **Обработка** — перекодирование в AV1 (SVT-AV1), изменение скорости, разрешения, FPS; аудиоэффекты. Двойной клик по файлу в очереди — кодировать его отдельно.
- **Изображения** — AVIF/WebP/PNG/JPG/ICO, лимит разрешения и размера файла, опция «Перезаписывать исходник» (удаляет оригинал, оставляя только сжатую версию — по умолчанию выключено).
- **Фото** — объединение изображений.
- **Base64** — кодирование/декодирование, опция «Создавать .txt файл» на выходе.
- **Промпт** — менеджер промптов с выбором файла.
- **⚙ Настройки** — Discord-канал, локальный сервер для расширения, обновление yt-dlp, автообновление программы.

## Автообновление

Программа сама проверяет GitHub Releases. Если bin (ffmpeg/yt-dlp/deno) не менялся — качается только лёгкий `update`-архив (десятки МБ) вместо полного (сотни МБ); если менялся — полный. Старые версии и установки без манифеста всегда получают `full`-архив.

## Обновление yt-dlp

Если перестало качать с YouTube/TikTok — открой **⚙ Настройки → Обновить yt-dlp**. Обновится `bin/yt-dlp.exe` без переустановки программы.

## Сборка из исходников

Нужен Python 3.10+ и зависимости:

```bash
pip install -r requirements.txt
```

Запуск из исходников:

```bash
python main.py
```

Сборка в .exe (Windows):

```bash
build.bat
```

Результат — папка `dist/SI-HYX vX.Y.Z/` и три готовых ассета для релиза (`…-full.zip`, `…-update.zip`, `manifest.json`). Имена и состав формирует `build.bat` автоматически.

> Папка `bin/` (ffmpeg, ffprobe, yt-dlp, deno) намеренно **не** хранится в репозитории из-за размера — её нужно положить рядом самому или взять из релиз-архива. Скачать:
> - ffmpeg/ffprobe: https://www.gyan.dev/ffmpeg/builds/ (ffmpeg-release-essentials)
> - yt-dlp: https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe
> - deno: https://github.com/denoland/deno/releases

## Структура кода

| Файл | Назначение |
|------|-----------|
| `main.py` | главное окно, точка входа, HTTP-сервер для расширения |
| `config.py` | импорты, пути, константы, стили |
| `utils.py` | утилиты (ffmpeg/ffprobe, cookies, deno) |
| `widgets.py` | кастомные виджеты и превью |
| `workers.py` | фоновые потоки: загрузка (yt-dlp) и обработка (ffmpeg) |
| `tabs.py` | вкладки интерфейса |
